# AI Resume Analyzer

An AI-powered Resume Analyzer built with **Python**, **Streamlit**, and **Google Gemini AI**.

Upload a PDF resume and receive instant AI-powered feedback including:

- 📄 Professional Summary
- ✅ Strengths
- ⚠️ Weaknesses
- 📊 ATS Score
- 🛠 Missing Skills
- 💡 Suggestions for Improvement

## Installation

Clone the repository

```bash
git clone https://github.com/prithvisinghtiku/AI-Resume_Analyzer.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```text
GEMINI_API_KEY=YOUR_API_KEY
```

Run

```bash
streamlit run app.py
```
